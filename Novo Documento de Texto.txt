Criar um comparativo a nível de código entre a implementação do Vue e do React. Itens para serem comparados:

- Como criar um componente?
React - 2 componentes criados MyButton(botão customizado) e UserItem(item da lista de usuários)
Vue - 2 Componentes criados MyButton(botão customizado) e UserItem(item da lista de usuários)
- Como usar CSS?
React - CSS importado na home e no componente de botão
Vue - CSS em arquivo separado no componente de botão e css no arquivo vue na home
- Como criar rotas?
React - Rotas são criadas no arquivo router, usando a lib react-router
Vue - Rotas são criadas no arquivo index, usando a lib vue-router
- Como controlar as interações dos componentes?
React - Usando callbacks exemplificados na home
Vue - Usando callbacks exemplificados na home
- Como passar dados de uma API para o componentes e/ou página?
React - Usando axios para chamadas web, atualizando o estado após o término da chamada.
	No caso dos componentes, passando via props, demonstrado na página de api.
Vue - Usando axios para chamadas web, atualizando o estado após o término da chamada.
	No caso dos componentes, passando via props, demonstrado na página de api.
- Como criar uma páginas?
React - Criar um componente normal, mas cadastra-lo como uma page na parte de router.
Vue - Criar um componente normal, mas cadastra-lo como uma page na parte de router.
- Qual a diferença do scaffolding gerado pelo os geradores de projeto de cada um deles?

O scaffolding do vuejs oferece uma maior variedade na criação,dando oportunidade de adiçao de algumas bibliotecas que são usadas no desenvolvimento
como vue-router e vuex(controle de estado), no caso do react não existe esta opção.

Isso tudo deverá ser feito para cada uma das bibliotecas, ou seja, Vue e React. Deverá ser entregue via Github, usando um repositório.
